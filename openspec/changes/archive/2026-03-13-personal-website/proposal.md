## Why

As a developer, I need a simple personal website to showcase my professional identity and resume. This establishes an online presence with a clean, modern look that reflects my work and background.

## What Changes

- Create a new personal website built with Next.js
- Simple, minimal design focused on clarity
- Resume presented as an interactive timeline
- Dark and light mode support
- Each page will be designed and built in separate changes

## Capabilities

### New Capabilities
- `site-foundation`: Next.js project setup, routing, global layout, dark/light mode toggle, and shared styling
- `resume-timeline`: Resume displayed as a vertical timeline showing experience, education, and skills in chronological order
- `homepage`: Simple landing page with name, title, and navigation to resume

### Modified Capabilities

## Impact

- New project — no existing code affected
- Built with Next.js (App Router)
- Dark/light mode via CSS variables or Tailwind dark mode
- No external API dependencies in this phase
- Individual pages (portfolio, contact, etc.) will be scoped to future changes
